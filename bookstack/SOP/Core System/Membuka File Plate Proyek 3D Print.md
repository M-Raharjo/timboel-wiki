**Tujuan:**  
Membuka file plate proyek 3d Print

**Trigger:**  
Kebutuhan untuk membuka file plate proyek 3d print

**Input:**  
- file plate proyek 3d

**Langkah:**  
1. Cek desktop komputer (pencet tombol `Win + D`), layar pojok kanan bawah. Cari file yang memiliki nama yang sama di judul dokumen fisik/cetal/print checklist komponen print. Jika ketemu, **lanjut ke langkah 6**
2. Buka web browser di komputer 3D printer. Akses [Google Drive Proyek 3D Print](https://drive.google.com/drive/folders/1eSAQJrRWF6qD_M45ju9KKmKAiFJgmxj6)
3. Cari folder nama proyek/kode komponen 3d print yang akan dikerjakan. Buka folder. Contoh Folder: `Mind in Motion`, folder yang dibuka adalah yang bernama `Mind in Motion`.
4. Buka sub-folder dengan judul `nama proyek - plate`. Contoh: `Mind in motion` sub foldernya `Mind in motion - plate`

  - **APABILA MELANJUTKAN PROSES PRINT** Terkadang file plate sebuah proyek print ada banyak. Cek judul `Dokumen Checklist Print` untuk memastikan
    - Contoh 1: Judul di Dokumen Checklist Print adalah `MIM Mind in Motion 1-79`, cari file plate yang memiliki tulisan `1-79`
    [![](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/scaled-1680-/WajjpoB8C2wjT3i8-image-1777305532564.png)](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/WajjpoB8C2wjT3i8-image-1777305532564.png)
    - Contoh 2: Judul di Dokumen Checklist Print adalah `TOR High Poly`, cari file plate yang mimiliki tulisan `HP` atau `High Poly`
    [![](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/scaled-1680-/oTPf17hkKVuvAwDB-image-1777305756970.png)](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/oTPf17hkKVuvAwDB-image-1777305756970.png)
  - **APABILA MEMULAI PROSES PRINT BARU** Pilih file yang tidak memiliki `Dokumen Checklist Print`
 
5. Unduh file dengan ekstensi `.gcode.3mf` di folder tersebut
5. Pindahkan hasil unduh file ke desktop komputer. Taruh di **pojok kanan bawah** tampilan desktop
6. Buka aplikasi Bambu Studio,
7. Layar pojok kiri atas, klik `FILE`, pilih `open project`
8. Pilih file yang akan diproses (file yang di desktop atau file yang diunduh)

**Output:**  
File plate proyek print berhasil dibuka
