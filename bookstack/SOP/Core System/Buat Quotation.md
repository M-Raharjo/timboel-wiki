**Tujuan:**  
Membuat file PDF quotation untuk dikirim ke customer

**Aturan:**
- Jangan memberikan diskon di Quotation
- Diskon hanya ada di invoice

**Langkah:**  
1. Buka [kirun database](https://kirun.pttimboel.com/app/home)
2. Ctrl + G -> New Quotation
3. Isi data customer
4. Tentukan valid till (rekomendasi: 2 minggu)
5. Pastikan Currency IDR
6. Masukan permintaan barang dengan ID barang
7. [Estimasikan Charges](https://wiki.pttimboel.com/link/496)
8. Save
9. Cek ulang
10. Submit
11. Print as PDF

**Output:**  
File PDF quotation
