**Tujuan:**  
Mempersiapkan dokumen panduan untuk mencetak plate sebuah proyek print

**Trigger:**  
Plate komponen sudah tersedia di dalam folder penyimpanan proyek print terkait

**Input:**  
- Akses ke Google Drive [Proyek 3D printing](https://drive.google.com/drive/folders/1eSAQJrRWF6qD_M45ju9KKmKAiFJgmxj6)
- File dokumen [Checklist Komponen Print](https://docs.google.com/spreadsheets/d/1vzbdZO4tGH5JHW_-cfYLRN2pEg9O1qb7l1KXDZBLXhY/edit?gid=0#gid=0)
- Akses [tracking 3D printing](https://docs.google.com/spreadsheets/d/1fsVSR_fHL2AxGm0g35uOucx71AizGN1MJaRNfw7o-mY/edit?gid=610922543#gid=610922543) di bagian `sheet 3D Printing`

**Langkah:**  
  1. Cari dokumen cetak/fisik/print `Checklist Komponen Print` yang sedang digunakan, ciri-ciri: kolom `Selesai?` nya belum selesai diisi oleh tanda centang, dan mesin printer sedang bekerja.
  - Contoh Aktif Digunakan, informasi belum lengkaip:
  [![](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/scaled-1680-/H2lGeX8sksSlPjkS-image-1777516827706.png)](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/H2lGeX8sksSlPjkS-image-1777516827706.png)
  - Contoh Selesai Digunakan, semua kolom terisi:
  [![](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/scaled-1680-/D1hBx1d57wf2cO6y-image-1777517744596.png)](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/D1hBx1d57wf2cO6y-image-1777517744596.png)
2. Apabila ada Checklist Komponen Print dalam bentuk fisik/cetak/print yang Aktif Digunakan, pakai dokumen itu sebagai panduan proses print. Hiraukan semua langkah `1. Menyiapkan Dokumen Cetak Plate` selanjutnya, dan lanjut ke tahap [3. Mencetak Plate](https://wiki.pttimboel.com/books/alur-3d-printing/page/3-mencetak-plate) dengan membuka halaman tersebut. Jika tidak ada, lanjut ke langkah 3 di bawah.
3. Cetak file dokumen checklist komponen print sebanyak yang dibutuhkan
4. Cari nama proyek yang belum dimulai proses printnya di [tracking 3D printing](https://docs.google.com/spreadsheets/d/1fsVSR_fHL2AxGm0g35uOucx71AizGN1MJaRNfw7o-mY/edit?gid=610922543#gid=610922543) di bagian `sheet 3d printing` (bagian bawah layar) dengan label kolom `plate` berisi `completed` dan kolom `print` berisi `menunggu`. 
  - Contoh: Monolith Ilham status `plate` nya `selesai` dan `print` nya `dikerjakan`, artinya Monolith Ilham sedang dikerjakan (ada dokumen fisik checklist komponen print).
  - Contoh: Squatling Hyogis status `plate` nya `selesai` dan `print` nya `menunggu`, artinya Squatling Hyogis siap di-print tapi belum dimulai proses printnya (belum punya dokumen fisik checklist komponen print).
[![](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/scaled-1680-/MbjNvvuOnxBTCM6a-image-1777429996814.png)](https://wiki.pttimboel.com/uploads/images/gallery/2026-04/MbjNvvuOnxBTCM6a-image-1777429996814.png)
5. [Buka file plate](https://wiki.pttimboel.com/books/core-system/page/membuka-file-plate-proyek-3d-print) menggunakan aplikasi Bambu Studio
6. Isi data nama proyek dan kode komponen (pakai nama file plate) pada baris `nama file print patung` di dokumen checklist komponen print
7. Tulis semua nomor plate di dalam file plate pada dokumen checklist komponen print. Contoh: File dengan nama `MIM - 001-120` dia punya 120 plate. Tulis angka 1 sampai 120 di kolom `Plate` dokumen fisik `Checklist Komponen Print`

**Output:**  
Dokumen checklist komponen print berisi data proyek dan daftar plate yang akan dicetak. Contoh:
