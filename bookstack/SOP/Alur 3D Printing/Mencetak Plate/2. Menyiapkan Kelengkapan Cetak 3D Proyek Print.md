**Tujuan:**  
Untuk memastikan peralatan yang dibutuhkan selama proses cetak tersedia

**Aturan**
- Bahan dan material diambil dari ruang 3D dan Gudang
- Permintaan ke Gudang wajib menyertakan nama proyek sebagai dokumentasi Gudang

**Trigger:**  
Mendapat perintah cetak proyek 3D print

**Input:**  
- Kode Komponen Proyek Print (singkatan nama proyek print/patung). Bisa dilihat di judul file plate patung/proyek print. Contoh: MIM 001-120 kode komponennya MIM
- Bolpen (1 pcs)
- Spidol hitam (1 pcs)
- Sticky note 4x10 cm (1 pak)
- Lakban kertas (1 pcs)

**Langkah:**  
1. Siapkan wadah kardus untuk menampung komponen print
2. Tempel label dari lakban kertas, tulis kode komponen proyek print pada wadah menggunakan spidol
3. Siapkan bolpen, spidol hitam, dan sticky note
5. Taruh semua barang yang disiapkan di ruang 3D Printer

**Output:**  
Ruang 3D printer memiliki kelengkapan untuk proses Proyek Print
