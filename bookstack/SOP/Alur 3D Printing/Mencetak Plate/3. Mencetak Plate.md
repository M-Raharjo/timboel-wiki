**Tujuan:**  
Mencetak komponen 3D

**Trigger:**  
Dokumen Fisik Checklist Komponen Print proyek print tersedia

**Input:**  
- Dokumen fisik checklist komponen print proyek print
- file plate proyek print

**Langkah:**  
1. [Buka file plate](https://wiki.pttimboel.com/books/core-system/page/membuka-file-plate-proyek-3d-print) proyek print sesuai yang tertulis di checklist komponen print
2. Persiapkan printer untuk proses cetak mengikuti panduan [Mengecek Proses Kerja Printer](https://wiki.pttimboel.com/books/core-system/page/mengecek-proses-kerja-printer)
3. Cetak komponen print mengikuti panduan [Mencetak Komponen Proyek 3D print ](https://wiki.pttimboel.com/books/core-system/page/mencetak-komponen-proyek-3d-print)
4. Ulangi dari langkah dua hingga seleuruh plate selesai dicetak

**Output:**  
Hasil cetak komponen proyek print dikumpulkan dalam 1 wadah dan siap dirangkai
