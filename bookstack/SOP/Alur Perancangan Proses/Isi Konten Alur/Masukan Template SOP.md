**Tujuan:**  
Memiliki Halaman SOP yang Rapi dan Terstandarisasi

**Template:**
```
**Tujuan:**  
Contoh Tujuan

**Aturan:**
- Contoh aturan 1
- Contoh aturan 2
- Contoh aturan 3

**Trigger:**  
Contoh Trigger  

**Input:**  
- Contoh input 1
- Contoh input 2
- Contoh input 3

**Langkah:**  
1. Langkah Pertama
2. Langkah Kedua
3. Langkah Ketiga

**Output:**  
Contoh Output SOP
```

**Aturan:**
- Hapus template yang tidak relevan
- Langkah harus selalu menggunakan bullet point angka
- Input dan aturan harus selalu menggunakan bullet point standar
- Pastikan langkah2 menggunakan bahasa imperatif (perintah)
- Di setiap awal bullet point menggunakan huruf besar

**Output:**  
Halaman SOP yang rapi dan disiap di isi
